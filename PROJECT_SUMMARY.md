## 🔧 Technologies & Algorithms

### Core Technologies
- HTML5 (Canvas API, Audio API, MediaDevices API)
- CSS3 (Grid, Flexbox, Animations, Gradients)
- Vanilla JavaScript (ES6+)
- Local Storage API

### Algorithms Implemented
1. **Maze Generation** - Depth-First Search (DFS)
2. **Flood Fill** - Recursive algorithm
3. **3D Projection** - Perspective projection math
4. **Particle Physics** - Velocity, gravity simulation
5. **Audio Synthesis** - Web Audio API oscillators

7. **Collision Detection** - AABB for maze walls
