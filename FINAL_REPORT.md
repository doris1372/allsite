## 🔧 Technologies & Algorithms

### Core Technologies
- HTML5 (Canvas API, Audio API, MediaDevices API, Screen Capture API)
- CSS3 (Grid, Flexbox, Animations, Gradients, Custom Properties)
- Vanilla JavaScript (ES6+)
- Local Storage API
- Web Audio API

### Algorithms Implemented
1. **Maze Generation** - Depth-First Search (DFS)
2. **Flood Fill** - Recursive algorithm for pixel art
3. **3D Projection** - Perspective projection mathematics
4. **Particle Physics** - Velocity, gravity, collision simulation
5. **Audio Synthesis** - Web Audio API oscillators
6. **WPM Calculation** - (characters/5) / minutes
7. **Collision Detection** - AABB for maze walls
8. **QR Code Generation** - Pattern generation algorithm
9. **Markdown Parsing** - Custom markdown to HTML parser
10. **Waveform Visualization** - Audio frequency analysis
